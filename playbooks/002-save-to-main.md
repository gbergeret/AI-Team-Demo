# 002 - Save to main

Save your work to the cloud. This hides git behind one friendly verb. Run it
when the person says "save to main" (or just "save").

## What it does
Takes everything that changed and backs it up to GitHub on `main`, so it is safe
and available on any device.

## Steps
1. Stage all changes.
2. Commit them. Use a short, clear automatic message, or, if the person offered
   a note, use their one line.
3. Push to `main`.

## After
Confirm in one line: "Saved to main. Your latest changes are backed up."
If there was nothing to save, say so plainly instead.
